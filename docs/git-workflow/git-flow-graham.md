\## 3. 操作流程



本次练习包含以下步骤：



1\. clone 远程仓库到本地；

2\. 创建 feature 分支；

3\. 在 feature 分支上新增文档；

4\. 使用 git add、git commit、git push 提交到远程；

5\. 在 GitHub 上创建 Pull Request；

6\. 邀请组员 review；

7\. review 通过后合并到 main；

8\. 删除本地和远程 feature 分支。
