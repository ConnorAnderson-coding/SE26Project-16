# 余弦相似度阈值实验

## 1. 实验目标

在生产用稠密向量模型与当前 `search_text` 拼接下，测量各意图 query 相对全部活动的 ES kNN `_score` 分布，据此选定语义硬截断阈值 `app.elasticsearch.semantic-absolute-threshold`（仅作用于「无 BM25 命中」的语义候选）。

## 2. 实验环境与方案

- 索引：`campus_activities`，活动数 **18**（seed 全量非 draft）
- 嵌入模型：`campus_gte`（thenlper/gte-small-zh，512-d，无 E5 query/passage 前缀）
- 向量字段：`activity_embedding`，`similarity: cosine`
- 分数映射：ES kNN `_score ≈ (1 + cosine) / 2`；表中 `approx_cosine = 2*score - 1`
- 文档侧向量化文本 `search_text` = `title` + `description` + `category`（含中文类别名）+ `tags` + `location`
- **不含**：`college`（避免「计算机学院」误召回）、活动记录、评价、报名/收藏计数
- 查询侧：对原始 keyword 做 `text_embedding`，再 kNN （`k=20`，覆盖全库排序）
- 复现：`python backend/scripts/cosine-threshold-experiment.py`（可选 `--rebuild` 先全量重建索引）
- 本报告阈值结论：**保持 τ=0.90**（不再下调/上调）

按意图分四类：完全匹配（exact）、近义（synonym）、主题（theme）、几乎无关（unrelated）。
